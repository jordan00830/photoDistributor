## Photo Distributor

## Setup dev env

1. [Download wxPython Here](http://www.wxpython.org/download.php)
2. install py2app

	```
	$ sudo pip install -U py2app
	```

## Test & Dev

  ```
  $ python photoDistributor.py
  ```

## Build mac executable file

  ```
  $ python setup.py py2app
  ```
